상담 전환 리포트 기준

배포 후 3~7일 동안 GA4 이벤트를 수집한 뒤 아래 비율로 상담창 성과를 판단합니다.

수집 기간

최소 3일, 권장 7일. 유입이 적은 날은 판단하지 말고 누적 기준으로 봅니다.

텔레그램 이동률

consult_telegram_click / consult_open

5% 미만이면 버튼 위치나 문구를 개선합니다. 5~15%는 정상, 15% 이상은 우수합니다.

페이지별 상담 오픈

consult_open + page_path

상담창이 많이 열리는 페이지의 CTA와 업체 안내를 우선 보강합니다.

공식주소 이동률

consult_official_click / consult_code_copy

코드는 복사했지만 이동이 낮으면 세이프클릭 문구와 공식주소 버튼을 보강합니다.

FAQ 의존도

consult_faq_open / consult_open

FAQ 클릭이 높으면 상담 첫 화면에서 해당 FAQ 주제를 더 위로 올립니다.

GA4에서 확인할 이벤트

  • consult_open
  • consult_provider_select
  • consult_code_copy
  • consult_official_click
  • consult_telegram_click
  • consult_faq_open
  • safe_click_open / safe_click_confirm
  • inquiry_builder_copy
  • event_checker_submit / payout_kit_submit
운영 순서: 1) 3~7일 데이터 수집 → 2) 상담 버튼 문구별 클릭률 확인 → 3) 낮은 전환 구간 보강 → 4) 오늘의 확인판 문구 업데이트 → 5) FAQ 순서 조정.
