/tools/는 검색 실행, 공개 근거 추출, 리스크 점수 기준 확인처럼 도구 자체를 빠르게 여는 허브입니다. 제보와 결과 로그는 커뮤니티 허브에서 따로 관리합니다.

도구 허브의 역할

  • 검색 조합을 바로 실행합니다.
  • 공개 글 URL에서 근거만 추출합니다.
  • 리스크 점수 기준과 해석 레이어를 확인합니다.
  • 클러스터 힌트와 최근 조회 이력을 비교합니다.

핵심 도구

도구 허브는 기능 실행 중심으로, 커뮤니티 허브는 검토 흐름 중심으로 나눴습니다.
