# 88ST 관리자 페이지 정적 초안

이 폴더는 `88st.cloud/admin/` 경로에 올릴 정적 관리자 화면 초안입니다.

## 포함 파일
- `index.html`: 관리자 메인 화면 초안
- `style.css`: 밝은 톤 유리광 계열 스타일
- `app.js`: 메뉴 전환, 프로모 검색용 샘플 스크립트

## 중요한 점
- 이 화면은 **정적 UI 초안**입니다.
- 실제 저장/수정/로그 조회는 Worker의 `/admin-api/*` 에서 처리하는 구조를 권장합니다.
- 현재 값은 데모 데이터이며, 실제 운영값과 연결되어 있지 않습니다.

## 추천 배치
- 사이트 저장소: `admin/`
- 공개 경로: `https://88st.cloud/admin/`

## 추천 보안
- `88st.cloud/admin/*` 는 Cloudflare Access 또는 별도 관리자 세션 인증 뒤에 두는 것을 권장합니다.
