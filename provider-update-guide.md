# 헤더 상담센터 전환 및 모달 축소 v6 (2026-05-12)

## 반영
- 헤더 메뉴를 `메인 / 블로그 / 도구 / 상담센터` 기준으로 통일
- `/guaranteed/` 페이지는 삭제하지 않고 유지
- `/guaranteed/`는 sitemap/SEO 유지
- 보증업체 직접 노출은 자동상담, 상담 페이지, 푸터, 본문 CTA 중심으로 유지
- 기존 대형 제휴 모달을 compact 가입 안내 모달로 교체
- compact 모달은 홈/블로그/도구에서만 노출
- `/guaranteed/`에서는 중복 모달 노출 제외
- 모달 카드 내용은 업체명, 가입코드, 코드 복사, 상담 보기 중심으로 축소
- 메인/블로그/상담센터에 `상담이 필요한 순간` 섹션 추가

## 상담 명분 설계
사용자가 상담할 이유를 명확히 만들기 위해 아래 상황을 전면 배치합니다.
1. 가입 전 코드·공식주소 확인
2. 첫충·매충·페이백 조건 확인
3. 출금 전 캡처 자료와 문의문 정리
4. 텔레그램 문의 문구 생성

## 운영 의도
헤더에서 보증업체를 제거해 광고 사이트 느낌을 줄이고,
상담센터와 도구를 중심으로 정보형 사이트 인상을 강화합니다.
전환은 자동상담/세이프클릭/문의 문구 생성기가 담당합니다.
