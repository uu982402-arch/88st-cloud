# GA4 Conversion Events v16 기준 전환 이벤트 목록입니다. ## 핵심 이벤트 - `provider_code_copy`: 보증업체 코드 복사 - `provider_official_click`: 공식주소 접속 클릭 - `provider_consult_click`: 업체 상담 확인 클릭 - `consult_open`: 자동상담 열기 - `inquiry_builder_copy`: 문의 문구 복사 - `safe_click_open`: 세이프클릭 안내창 열기 - `blog_provider_hub_click`: 블로그 하단 업체 허브 클릭 ## 볼 지표 1. `/guaranteed/` 방문 대비 코드 복사율 2. 코드 복사 대비 공식주소 클릭률 3. 블로그 방문 대비 업체 허브 클릭률 4. 상담 열기 대비 텔레그램 이동률 5. 문의 문구 생성 대비 복사율 ## 개선 기준 - 코드 복사율이 낮으면 카드 문구와 버튼 위치를 조정합니다. - 공식주소 클릭률이 낮으면 코드 복사 후 접속 안내 toast를 강화합니다. - 블로그 허브 클릭률이 낮으면 글 하단 CTA를 주제별로 더 좁힙니다.
