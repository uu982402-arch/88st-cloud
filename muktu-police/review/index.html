이 페이지는 실제 검토 순서를 문서형으로 보여주는 안내 페이지입니다. 접수, 검색, 조회, 기록의 네 단계를 명확하게 분리했습니다.

필수 자료

  • 사이트명 / 도메인
  • 검색 결과 URL
  • 공지나 안내문 URL
  • 입출금 캡처 / 대화 내역
  1. 1. 접수제보 내용과 공개 링크, 캡처 자료를 먼저 모읍니다.
  2. 2. 검색사이트명과 도메인을 나눠 검색하고 공개 검색 흔적을 확인합니다.
  3. 3. 조회도메인·IP 조회 결과를 붙여 등록일, DNS, ASN, PTR을 확인합니다.
  4. 4. 기록핵심 근거만 짧게 정리해 상태와 함께 결과 로그에 남깁니다.
