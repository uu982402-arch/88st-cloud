DOMAIN + IP LOOKUP

도메인과 IP를 점수·해석까지 같이 봅니다.

값만 나열하지 않고 등록일, DNS, ASN, PTR을 바탕으로 짧은 해석과 점수를 같이 보여줍니다.

도메인등록일 · DNS · 점수기술 신호 기반 1차 판정IPASN · PTR · 해석조직·네트워크 힌트 확인근거 추출공개 URL 정리검색 후 근거를 메모할 때 사용

먼저 보는 기준

  • 신규 생성 도메인은 보수적으로 봅니다.
  • 만료 임박, A 레코드 부재, 네임서버 부재는 점수에 바로 반영합니다.
  • ASN · 대역 · 최근 조회 겹침은 군집 힌트로 봅니다.

도메인 조회

등록일, 만료일, DNS, 네트워크 힌트와 점수 해석을 함께 불러옵니다.

도메인을 입력해 주세요.예: example.com

IP 조회

ASN, 조직, PTR과 해석 레이어를 같이 보여줍니다.

IP를 입력해 주세요.예: 1.1.1.1
