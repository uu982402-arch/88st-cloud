DOMAIN + IP LOOKUP

도메인과 IP를 나눠 조회하고
리스크 점수까지 같이 봅니다.

등록일, 만료일, 네임서버, A 레코드, ASN, PTR을 바탕으로 1차 점수와 해석 레이어를 제공합니다. 점수는 기술적 신호 기준이며 최종 판단은 공개 검색과 함께 보세요.

도메인등록일 · DNS · 점수기술 신호 기반 1차 판정IPASN · PTR · 해석조직·네트워크 힌트 확인근거 추출공개 URL 정리검색 후 근거를 메모할 때 사용

해석 기준

  • 신규 생성 도메인은 기본적으로 보수적으로 봅니다.
  • 만료 임박, A 레코드 부재, 네임서버 부재는 점수에 바로 반영합니다.
  • 복수 ASN·대역, 최근 조회 이력과의 일치도는 군집 힌트로 봅니다.

도메인 조회

도메인을 입력하면 등록일, 만료일, DNS, 네트워크 힌트와 점수 해석을 불러옵니다.

도메인을 입력해 주세요.예: example.com

IP 조회

IP를 입력하면 ASN, 조직, 위치 힌트, PTR과 해석 레이어를 보여줍니다.

IP를 입력해 주세요.예: 1.1.1.1

같이 보면 좋은 글

점수와 값만 보지 말고 왜 이런 값을 보는지 기준부터 이해하는 편이 좋습니다.
