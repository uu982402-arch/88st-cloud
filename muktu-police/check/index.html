DOMAIN + IP LOOKUP

도메인과 IP를 나눠 조회하고
공개 검색으로 다시 확인합니다.

도메인 생성 시점, 만료일, 네임서버, A 레코드, ASN, PTR 같은 기본 정보를 한 화면에 모았습니다. 검색창에서 본 내용과 연결해 비교하기 좋게 정리했습니다.

도메인등록일 · DNS도메인 기준 조회IPASN · PTRIP 기준 조회구글링공개 검색검색 흔적과 함께 보기

실전 체크

  • 도메인이 너무 최근 생성인지 확인합니다.
  • 네임서버와 A 레코드가 과하게 흔들리는지 봅니다.
  • 같은 IP/ASN 힌트가 있으면 공개 검색으로 다시 확인합니다.

도메인 조회

도메인을 입력하면 등록일, 만료일, DNS, 네트워크 힌트를 불러옵니다.

도메인을 입력해 주세요.예: example.com

IP 조회

IP를 입력하면 ASN, 조직, 위치 힌트, PTR 응답과 검색 조합을 보여줍니다.

IP를 입력해 주세요.예: 1.1.1.1
