먼저 확인할 항목

  • 같은 네트워크를 공유하는 환경이 있었는지 확인합니다.

  • 보너스 이용 중이라면 약관상 중복 계정·공유 환경 항목을 다시 확인합니다.

왜 바로 대응이 필요한가

  • 보너스나 출금 단계에서 추가 확인이 붙을 수 있어 초기 기록이 중요합니다.

운영 팁

  • 임의로 반복 접속을 바꾸기보다 현재 환경을 먼저 정리하는 편이 좋습니다.
주의할 점
